# guia-estudos-matematica-tecnica-demostracao-notebook-lm
Este repositório reúne estudos sobre lógica formal e técnicas de demonstração (Indução, Contraposição, Absurdo) aplicadas à Ciência da Computação. Inclui teoria baseada em Judith Gersting, análise de invariantes de laço e implementações práticas de algoritmos para validar a corretude de software.
